# 搜索引擎

## Reds

> [Reds](https://github.com/tj/reds) 是一个轻量的基于NodeJS和Redis的搜索引擎，由TJ Holowaychuk 开发，这个模块原本是为优化 Kue 搜索能力而开发，但是它也非常适合作为轻量的通用搜索库而加入到Blog、文档系统中去。

```bash
$ npm install reds
```
